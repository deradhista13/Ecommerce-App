## Getting Started

1. Clone repo:
```sh
   git clone https://github.com/deradhista13/Ecommerce-App.git
   ```
2. Install package
```sh
   npm install
   ```
3. Run development server
```sh
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
